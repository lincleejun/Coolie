当前项目是干什么的
The user is asking about what the current project does. I can answer by reading CLAUDE.md and the top-level directory structure, then summarize the architecture and dev workflow.

根据项目的 CLAUDE.md，这个项目是 MAG（Multi-Agent Generation），一个智能视频模板生成系统。

核心功能：

  • 接收用户素材（图片、视频等），自动生成专业视频模板
  • 使用多个 AI Agent 协同工作，通过 LangGraph 进行编排
  • 底层跑多个 LLM（Gemini / OpenAI / 字节 ARK）

技术栈：

  • 后端：Python / FastAPI，Agent 自动发现机制，WorkflowPipeline 驱动执行
  • 前端：React 19 + TypeScript，Rush.js monorepo
  • 存储：TOS（字节对象存储）
  • 监控：LangSmith + Fornax

部署架构上有两个独立 FaaS 服务：

  1. lib/mag-http2rpc-gateway/ — HTTP 总入口（ /mae/generate 等）
  2. project_mag/microservice/ — 内网 SDK / Thrift 封装层，供本地开发调用

开发方法论以 Eval-Driven Development 为核心：用 eval 定义问题，AI 跑修复循环，CLI 是 uv run mae eval/submit。
